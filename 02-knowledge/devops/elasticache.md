# Elasticache

In-memory cache web service that makes its easy to deploy, operate and scale
Allows to retrieve information from fast, managed, in-memory caches instead of disk-based databases 
Stores the queries instead of hitting database
Used to increase database and web application performance

**Supports**
1. **Memcached**
- Scales horizontally

1. **Redis**
- Multi-AZ
- Can do backups and restores
