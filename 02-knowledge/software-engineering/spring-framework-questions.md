# Spring Framework Questions

**What is Spring Framework ?**

Development framework for enterprise java (J2EE)
Spring is to make J2EE development easier to use and promote good programming practices

**What are the advantages of the Spring Framework?**

Lightweight
- Framework doesn’t need to be completely loaded to be useful
- Use 1 framework without using another 1

Flexible 
- Uses Dependency Injection
- Configured by XML or annotation-based style

Loosely coupled
- Dependency injection

Easy to test
- Provides testing modules that make testing easier

**Spring Vs Spring Boot**

Spring
- XML based configuration
- Application creation/ execution takes more time
- More boiler plate code

SpringBoot
- Annotation-based
- Application creation/execution is quicker
- Reduced amount of boiler plate code
