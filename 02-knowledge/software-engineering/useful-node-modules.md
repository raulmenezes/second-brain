# Useful Node modules

**PM2**
Use for calling large applications and infrastructure
**Process management** for production applications, and a **load-balancer** to help with any possible performance tweaks https://github.com/Unitech/pm2  **Browserify** Lets you require('modules') in the browser by bundling up all of your dependencies.
Allows us to create the kind of client-side codebase that’s organized, structured, and easy to navigate. https://github.com/browserify/browserify  **Async.js**
Rebuilt the way JavaScript content interacts with your web pages, allowing you to increase performance by getting rid of render-blocking JavaScript. Load JavaScript content alongside everything else https://github.com/caolan/async

**Commander.js**
Light-weight, expressive, and powerful command-line framework https://github.com/tj/commander.js  **Underscore.js**
More than 100+ helpers  https://github.com/jashkenas/underscore  **UglifyJS2**
Trims and minifies your JavaScript files for you automatically. https://github.com/terinjokes/gulp-uglify
