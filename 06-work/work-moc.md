# Work MOC

Central index for work-related notes that should stay separate from personal projects.

## Suggested Categories

- Active work notes
- Meetings
- Company notes
- Incidents
- Career notes

## Usage

- Put active work notes in `06-work/`.
- Keep reusable technical knowledge in `02-knowledge/`.
- Move outdated work notes to `05-archive/`.
