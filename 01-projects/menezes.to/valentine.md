# Valentine

A fun interactive website created for Valentine's Day.

## Purpose

- Build a playful seasonal experience with a strong emotional tone.
- Create something personal, memorable, and visually polished.
- Use it as a small creative experiment in interaction and animation.

## Features

- Animated landing page
- Interactive prompts or choices
- Personalized message flow
- Mobile-friendly presentation
- Shareable final screen

## Tech Stack

- Next.js
- Tailwind CSS
- Framer Motion

## Ideas

- Add branching interactions based on user choices.
- Include subtle audio or visual surprises.
- Personalize content with names, dates, or custom messages.
- Add a final reveal page with a keepsake-style layout.

## Future

- Reuse the concept for birthdays, anniversaries, or other events.
- Turn it into a template for themed one-page experiences.
- Add a tiny CMS or config file for quick yearly updates.
